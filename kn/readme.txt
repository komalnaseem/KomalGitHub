A starter theme called Komal.

== Description ==

Hi. I'm a starter theme called Komal. I'm a theme meant for creating your very own job portfolio website. 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's kn.zip file. Click Install Now.
3. Click Activate to use your new theme right away.

==Custom Settings ==

//Custom Home Page//
When you go to the live site, the first page that you see is "Home" Page. The user can update the content in the post on the home page to their liking. The user can also customize the "home" page by following these steps:

1. Update the home page post content to their liking by clicking "Edit" on the page.

2. "Custom Fields" section (bottom left of the screen inside the post): Updating values in custom fields
	a.The user can update values in custom fields for availability, please make sure not to modify the custom field name as that has been set in the code.
		- Name: availability
		- Value: Enter a date here like "September 15, 2016"
		- Click on the "Update" button in order for the changes to take effect.
		Note: This custom field will be only displayed on the custom home page.
		
	b. The user can update the values in the custom fields for the mission statement, please make sure not to modify the custom field name.
		- Name: mission
		- Value: Enter your own mission statement here.
		- Click on the "Update" button in order for the changes to take effect.
		Note: This custom field will be only displayed on the custom home page.	
		
3. "Featured Image" section (bottom right of the screen inside the post):
	a. If there is no image and you would like to add a "Featured Image", then click on "Set featured image". There are also functionalities available to load an image from the site or load your own image, edit the image size, edit the image title and other information about the image.
	b. If there is an existing image and you like to edit or update the image then click on the image to edit or update. There are also functionalities available to load an image from the site or load your own image, edit the image size, edit the image title and other information about the image. 
	c. If you do not want a "Featured Image", then click on the "Remove Featured Image" link. 

4. "Page Attributes" section (on the top of the "Featured Image" section inside the post):		
	There is no need to update any option here, the user should leave everything as default.
	
5. The "Template" for the custom home page is set to 'Custom Home Page'. This new custom template is created solely for the purpose of the home page, and is different from all the other pages on the site. 
     Note: The user can change the 'Custom Home Page' template to the 'Default Template' if they do not wish to have a custom home page. 
	
6. Under the "Publish" section (top right of the screen inside the post), there are also functionalities available to:
	a.	Click on "Preview" if you would like to preview your custom post before publishing it.
	b.  All other options can be left as default.
	c.	Click on "Update" if you would like to "Update" the changes on your custom post.
		
7.  Under the "Edit Page", you will see "page updated. View Page", then click on "View Page" to see your changes.

// Portfolio Options Page//

The "Portfolio Options" page could be found on the main menu under the "Appearance" section. Click on the "Portfolio Options" page and it will present you with the "Job Portfolio Options Page" title and some details about the available options. This page includes three customizable options that the user can include on their live site. 

Option 1. A textbox - "Enter your email address: " to display your email address. This allows the user to simply type in their email address and it will be displayed on the footer of the site. 

Option 2. A checkbox - "Include featured image: " for displaying the featured image. If the user checks the checkbox, then the featured image will be displayed on any page/post where the feautured image is selected.

Option 3. A radio button - "Choose the sidebar postion: ", this has two options which are "Left" or "Right". If the user chooses "Left", then the sidebar will move to left of the content/screen. If the user chooses "Right", then the sidebar will move to the right of the content/screen. 

Once all the above options are implemented, then click on "Save Changes" and view your live site to see their customizations in option.

Note: The user must type in information in the textbox area for the email, in order for it to show up on the live site. For example, if no email address is inserted, then no email address will be displayed on the live site. 

== Functionality ==

// Responsive Design //
This site presents responsive design for two smaller screen sizes as described below:

1. Screen size less then 750px:
	a. The menu will turn into a button.
	b. The sidebar will move before or after the content depending on the sidebar positioning seletion in the options page.
	c. The sidebar text size will reduce to 75% of the global setting.
	
2. Screen size less then 480px:
	a. 1a and 1b as mentioned above.
	b. The sidebar text size will reduce to 60% of the global setting.
	c. The mission statement will not be displayed on the custom home page.
